Crea una nueva instancia de GPIO y la inicializa.

Esta función crea una nueva instancia de una estructura GPIO, asignando los valores iniciales para el puerto y el pin especificados.

Parámetros
portNúmero de puerto GPIO.
bitNúmero de pin dentro del puerto GPIO.
Devuelve
gpio_t Retorna un puntero a la nueva instancia de la estructura GPIO.

Crea una nueva instancia de GPIO y la inicializa.

Esta función actúa como un equivalente a malloc para estructuras GPIO. Busca en un array estático predefinido la primera instancia que no esté en uso y la asigna para su uso. Este enfoque es útil en sistemas con memoria limitada donde la asignación dinámica (como con malloc) podría ser problemática. Esta función es específica para sistemas que definen USE_STATIC_MEM, permitiendo el uso de memoria estática en lugar de dinámica.

Nota
Esta función es parte de una implementación que usa memoria estática. Asegúrate de que USE_STATIC_MEM esté definido para habilitar esta función.
Devuelve
gpio_t Retorna un puntero a una instancia de GPIO si se encuentra una disponible. Retorna NULL si todas las instancias están en uso.
